# Day 14 - Monitoring Dashboard & API Health Check

## Objective

Implemented a monitoring dashboard for real-time fraud prediction analysis.

## Features

- Streamlit Monitoring Dashboard
- KPI Cards
- Prediction Log Viewer
- Fraud Rate Monitoring
- Average Confidence Score
- Pie Chart
- Histogram
- Prediction Trend
- API Health Endpoint

## API

GET /health

Returns:

{
  "status": "healthy",
  "model_loaded": true,
  "version": "1.0.0"
}

## Outcome

The project now supports production monitoring with dashboard analytics and health status reporting.
